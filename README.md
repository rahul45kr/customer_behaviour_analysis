# 🛒 Customer Shopping Behavior Analysis

## 📌 Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer segments, and key revenue drivers. The goal is to derive actionable insights that help businesses improve marketing strategies, customer retention, and overall sales performance.

---

## 📊 Dataset

* **Source**: (e.g., Kaggle / E-commerce dataset / Retail data)
* **Format**: CSV / SQL Database
* **Key Features**:

  * Customer ID
  * Age, Gender
  * Product Category
  * Purchase Amount
  * Purchase Date
  * Payment Method

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy) – Data Cleaning & Analysis
* **SQL (MySQL / PostgreSQL / SQL Server)** – Data Querying
* **Power BI** – Data Visualization & Dashboard
* **Gamma** – Presentation (PPT)

---

## 🔄 Project Workflow

### 1. Data Collection

* Imported dataset from source (CSV / Database)

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized formats (date, categorical values)

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified top product categories
* Studied purchase trends over time
* Compared spending behavior across segments

### 4. SQL Analysis

* Created tables and loaded data
* Wrote queries for:

  * Monthly sales trends
  * Top customers
  * Revenue by category
  * Customer segmentation

### 5. Data Visualization

* Built an interactive **Power BI dashboard** including:

  * Total Sales & Revenue KPIs
  * Sales by Category
  * Customer Segmentation (Age/Gender)
  * Time-based Trends
  * Top Customers

---

## 📈 Key Insights

* A small percentage of customers contribute to a large portion of revenue
* Certain product categories consistently generate higher sales
* Seasonal trends influence customer purchasing behavior
* Repeat customers have higher average purchase value

---

## 💡 Business Recommendations

* Target high-value customers with personalized offers
* Focus marketing on top-performing product categories
* Use seasonal trends for better inventory planning
* Improve retention strategies for repeat customers

## ▶️ How to Run

### 🔧 Prerequisites

* Python 3.x
* SQL Database (MySQL / PostgreSQL / SQL Server)
* Power BI Desktop

### ⚙️ Steps

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/customer-shopping-analysis.git
   ```

2. Install required libraries

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Run Python scripts for data preprocessing

4. Execute SQL queries in your database

5. Open the Power BI (.pbix) file to explore the dashboard.
